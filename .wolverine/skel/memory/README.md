# Memory Directory

This folder contains daily session logs and long-term memories.

## File Structure

```
memory/
├── 2026-03-07.md          # Daily session log (auto-created)
├── 2026-03-08.md
├── ...
└── long_term/              # Persistent memories (future feature)
    ├── facts.md
    ├── preferences.md
    └── skills.md
```

## What Gets Logged

### Every Session
- User requests and my responses
- Tools executed and results
- Decisions made and why
- Failures and recovery attempts

### Significant Events
- New skills created
- Patterns recognized
- Breakthroughs discovered
- Preferences learned

## Daily Log Format

```markdown
# YYYY-MM-DD

## Session Start
- **Time:** HH:MM
- **Mode:** {{MODE}}
- **Active Goals:** {{GOALS}}

## Key Events

### HH:MM - {{EVENT_TYPE}}
**What happened:**
{{DESCRIPTION}}

**Tools used:**
{{TOOLS}}

**Outcome:**
{{OUTCOME}}

**Lesson learned:**
{{LESSON}}

## End of Day Summary
- Tasks completed: {{COUNT}}
- Skills created: {{COUNT}}
- Failures: {{COUNT}}
- Insights: {{COUNT}}
```

## How I Use Memory

1. **Morning:** Read yesterday's log for context
2. **During session:** Append events in real-time
3. **Evening:** Summarize key learnings
4. **Heartbeat:** Check memory for patterns
