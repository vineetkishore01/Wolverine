# 🐺 WOLVERINE
### J.A.R.V.I.S.-Class Recursive Intelligence

Wolverine is a sovereign, self-evolving AI architect designed for systemic dominance on ultralight hardware. Unlike traditional reactive agents, Wolverine leverages a **Neural Engine** and **Hierarchical Memory** to transcend hardware limitations and achieve high-precision autonomy.

---

## 🚀 Core Engine Features

### 🧠 AGI Neural Engine (Phase 1+)
A central orchestration layer that routes intent through specialized cognitive centers. Wolverine doesn't just "chat"—it introspects, plans, and evolves.

### 📚 Hierarchical Memory System (HMS)
Total context awareness through a 5-layer retrieval architecture:
- **System**: Identity and Core Directives.
- **Session**: Immediate conversation flow.
- **Working**: Persistent multi-task scratchpad.
- **Semantic**: Relevant facts from the global brain.
- **Episodic**: Historical session insights and summaries.

### 🔄 Recursive Evolution Loop
- **Procedural Learning**: Automatically saves successful multi-step workflows.
- **Failure Analysis**: Learns from errors to ensure they are never repeated.
- **Self-Reflection**: Post-task meta-cognition to optimize future performance.

### 🛠️ Elite Tooling
- **Agentic Search**: Glob → Grep → Read hierarchy for surgical workspace exploration.
- **Capability Scanning**: Real-time discovery of tools, skills, and system powers.
- **Dynamic Error Recovery**: Self-correcting tool execution with learned recovery patterns.

---

## 🔋 Optimized for Ultralight Hardware
Wolverine is engineered to run on **4GB VRAM** GPUs using optimized local models like **Qwen 3 4B** or **Llama 3 8B**. The architecture is the intelligence—not just the weights.

## 🛠️ Getting Started

1.  **Configure Environment**: Set up your API keys and local model endpoints in `.env`.
2.  **Initialize Brain**: `npm install && npm run build`.
3.  **Deploy Agent**: `npm start`.
