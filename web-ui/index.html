1. Create an app + bot in Discord Developer Portal.
2. Enable required intents (including Message Content if needed).

5. Click Test, Save, then Send Test.

What is MCP?
Model Context Protocol (by Anthropic) is a standard that lets AI connect to external tools and data sources.

Each MCP server exposes a set of tools — Wolverine discovers them automatically when you connect. The AI can then call them during tasks, just like its built-in tools.

Servers run locally as child processes (stdio) or connect to remote HTTP endpoints (SSE). No data leaves your machine unless the tool itself calls an external API.
